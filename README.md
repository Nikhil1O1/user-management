# User Management


Welcome to the User Management repository! This project is built on node.js as a backend and it consist of api used for setting up a user managemnet in a website

## Getting Started

To set up the project and install all dependencies, follow these steps:

1. **Navigate to the Project Directory:**
   Move to the base location where `package.json` is present.

2. **Install Dependencies:**
   Run the following command to install all required dependencies:
   ```bash
   npm install
   ```

### Configuration

API keys are essential for the proper functioning of the program. Follow these steps to configure your API keys:

1. **Create a `config.env` file:**
   Create a file named `config.env` in the project root.

2. **Add Your API Key:**
   Inside `config.env`, add your API key like this:
   ```env
   API_KEY=your_api_key_here
   ```

   Make sure to replace `your_api_key_here` with your actual API key.
